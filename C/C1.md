## C1
### 1주차 1교시: C 언어 개요

**MASM**: 어셈블리어 문법 (Microsoft Assembler): 난해해서 고급언어를 만듦

**컴파일러**: C, C# -> `.exe`로 운용,   
**C** -> `CLR` (common Language Routine)로 운용  
**Java** -> `JVM` (Java Virtual Machine)로 운용
**인터프리터**: Python   

**ANSI**: American National Standard Institution

`Kernel`은 C를 이용해 구현됨 by 켄 톰슨, 테니스 리치  
`UNIX`는 C로 구성됨

**객체지향형**: C++   
**절차지향형**: C

A -> B -> C | 순서로 개발됨  
`ALGOL`: Algorithm Language   
`BCPL`: Basic Combined Programming Language  
`C`: BCPL + 자료형  

장점:  
> (1) 이식성  
> (2) 하드웨어 제어 [하드디스크, CD 등]  
> (3) 구문이 명확함  

`Compiler`: 소스코드(원시코드) -> 목적코드(프로그램이나 하드웨어가 처리하기 좋은 형태)
`Interpreter`: 소스코드 -> 목적코드(한줄씩 시행)

`Link`: 여러개의 목적코드를 하나로 합치는 과정.  
`Build`: Compile + Link 

C-Compiler: 개많음, 문법이 다 다름. 
> Visual C++, gcc, G++ etc......  
> `gcc`: GNU 프로젝트의 오픈소스 컴파일러

C언어의 `main()`함수: 컴파일러가 시행되면 프로그램이 시작하는 지점.  
- 함수는 하나의 기능을 가지고 있으며, 이 함수가 여러개 모이면 기능, 함수(기능)이 여러개 모인 것을 프로그램이라고 함.

C 표준 라이브러리를 사용할 경우 프로토타입이 정의되어있는 파일을 `#include` 해야함.


--- 
### 기본 문법

`void`: 값이 비어있음을 뜻함.  
`{}`: 기능의 시작과 끝을 뜻함  
`return`: Output 값을 지정.  
`#include`: same with `import`  
`<stdio.h>`: C 표준 라이브러리 중 하나. `printf` 함수

## C2
### 1주차 2교시
입출력 함수 = I/O 함수  
`keyword`의 뜻: 예약어

**a1. 반환문: Return**  
return 0; return 0은 프로그램이 성공하면 종료됨.  
return 1; return 1은 프로그램이 실패하면 종료됨.  

return 프로그램이 종료될 때 호출되는 부분으로 다시 돌아가서 종료됨.

**b2. 변수(variable)**
- C언어에서 데이터를 입력받으려면 입력받고자 하는 데이터의 유형을 해당 크기의 메모리에 **우선 할당**해야함.
- 데이터를 저장하기 위해 프로그램에 의해 이름을 할당받은 **메모리 공간**을 `변수`라고함.

### 기본 문법
`printf()`: print  
`scanf()`: 입력을 담당함. (input)  | scan formatted  
`&` 와 `%`: &는 **주소 연산자**: 입력받은 데이터를 변수에 저장하라는 의미; &는 **형식 지정자**: type annotation.  
`#include <stdio.h>`: 이건 헤더파일   
`//`: 주석  
`/**/`: 여러 줄 주석  
`\n`: 줄바꿈
