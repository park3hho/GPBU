# 6주차 1교시: 제어구조: 단순 for문
## V23. for문의 개념
###  (1) 반복문의 개념
> 인간은 반복을 싫어하지만 프로그램에서는 반복적인 작업들이 반드시 필요함

- 반복(iteration)은 꼭 필요함

### (2) 대표적인 반복문 for문과 while, do~while 문이 있음
- for문과 while문은 서로 변환이 가능하여 반복문을 작성할 때 어느 쪽을 선택해도 상관 없음
> for문 반복 횟수를 정확히 알고 있을 때, while 조건에 따라 결정되어, 횟수가 정확하지 않을 때

## W24. for문의 활용
### (1) for문의 사용법
```yaml
for (1초기화식; 2조건식; 4증감식) {
  3실행문장;
}
```
