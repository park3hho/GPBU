# 2주차 1교시: C언어 개요
## A1. printf() 함수의 기본 사용법
### (1) 표준 입출력 함수와 #include 지시문 개요
- C언어에서 표준 입출력 기능을 제공하는 핵심 헤더파일은 `stdio.h`임

#### #include 지시문 개요
- 다른 헤더 파일의 내용을 현재 소스 코드에 포함시키는 기능을 함.
- 사용자가 헤더파일을 직접 만들어서 포함시킬 수 있음.

#### 이스케이프 시퀀스(escape sequence)
- 줄바꿈`\n`, 탭`\t`, 큰따음표.
- `\"` 등의 특수 문자를 이스케이프 시퀀스라고 함

| **이스케이프 시퀀스** | **출력 내용**  | **이스케이프 시퀀스** | **출력 내용**  |
|-----------|--------|-----------|--------|
| \'        | 작은 따옴표 | \n        | 줄 바꿈   |
| \"        | 큰 따옴표  | \r        | 캐리지 리턴 |
| \?        | 물음표    | \t        | 수평 탭   |
| \\        | 백슬래시   | \v        | 수직 탭   |
| \a        | 경교음 발생 | \f        | 폼피드    |
| \b        | 백스페이스  |           |        |

## B2. `printf()`의 서식지정자
### 1. 숫자의 출력
- `printf()`는 괄호 안에 큰 따옴표로 감싸면 숫자도 무조건 문자로 취급함
- **서식문자-지정자**를 이용해 변수 내에 서식을 지정할 수 있음. (콤마로 구분함)
- 서식 앞에는 `%`가 붙음. 서식의 개수와 콤마 뒤의 파라미터 값이 같아야함.

```
printf("숫자를 입력하시오.: %d", 100)
```

#### ㄱ. 자주 쓰는 숫자 출력 서식
| **서식**    | **서식 설명**   | **서식 값의 예**     |
|-----------|-------------|-------------------------------------------|
| %d        | 4byte 정수    | \n                                        |
| %x        | 16진법        | printf("숫자를 입력하시오.: %x", 10) <br/> 출력: a  |
| %o        | 8진법         | printf("숫자를 입력하시오.: %x", 10) <br/> 출력: 12 |
| %f        | 실수          |  |
| %lf       | 8byte 실수    |  |
| %u        | 4byte 양수    |  |
| %hi, hu   | 2byte 정수 양수 |  |
| %li, lu   | 4byte 정수 양수 |  |
| %lli, llu | 8byte 정수 양수 |  |
| %llf      | 12byte 실수   |  |
| %e, E     | 부동소수점 e, E  |  |

### 2. 문자의 출력
| **서식** | **서식 설명**  | **서식 값의 예**  |
|------|------------|--------|
| \'   | 문자 출력      |       |
| \"   | 문자열 출력     |        |
| %p   | 포인터 주소값 출력 |      |

## C3. 자리수를 맞춘 출력

**정수**

| **서식** | **서식 설명**                           |
|--------|-------------------------------------|
| %d     | 숫자의 자릿수만큼 출력                        |
| %5d    | 오른쪽에 붙여서 정렬함, 5자리를 확보함              |
| %05d   | 오른쪽에 붙여서 정렬하고 빈칸을 0으로 채우고 5자리 확보  |

**실수**

| **서식** | **서식 설명**      |
|-------|----------------|
| %f    | 소수점 여섯 자리까지 출력 |
| %7.1f | 소수점 첫째 자리만 출력  |
| %7.3f | 소수점 셋째 자리만 출력  |

### 문자열형 데이터 서식은 오른쪽에 맞추어 출력됨
| **서식** | **서식 설명**   |
|--------|-------------|
| %s     | 문자열만큼 출력    |
| %10s   | 10칸 확보 후 출력 |
