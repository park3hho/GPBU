# 2주차 1교시: C언어
## A1. printf() 함수의 기본 사용법
### (1) 표준 입출력 함수와 #include 지시문 개요
- C언어에서 표준 입출력 기능을 제공하는 핵심 헤더파일은 `stdio.h`임

#### #include 지시문 개요
- 다른 헤더 파일의 내용을 현재 소스 코드에 포함시키는 기능을 함.
- 사용자가 헤더파일을 직접 만들어서 포함시킬 수 있음.

#### 이스케이프 시퀀스(escape sequence)
- 줄바꿈`\n`, 탭`\t`, 큰따음표.
- `\"` 등의 특수 문자를 이스케이프 시퀀스라고 함

| **이스케이프 시퀀스** | **출력 내용**  | **이스케이프 시퀀스** | **출력 내용**  |
|---------------|--------|-----------|--------|
| \\'           | 작은 따옴표 | \n        | 줄 바꿈   |
| \\"           | 큰 따옴표  | \r        | 캐리지 리턴 |
| \\?           | 물음표    | \t        | 수평 탭   |
| \\\           | 백슬래시   | \v        | 수직 탭   |
| \a            | 경교음 발생 | \f        | 폼피드    |
| \b            | 백스페이스  |           |        |

## B2. `printf()`의 서식지정자
### 1. 숫자의 출력
- `printf()`는 괄호 안에 큰 따옴표로 감싸면 숫자도 무조건 문자로 취급함
- **서식문자-지정자**를 이용해 변수 내에 서식을 지정할 수 있음. (콤마로 구분함)
- 서식 앞에는 `%`가 붙음. 서식의 개수와 콤마 뒤의 파라미터 값이 같아야함.

```
printf("숫자를 입력하시오.: %d", 100)
```

#### ㄱ. 자주 쓰는 숫자 출력 서식
| **서식**    | **서식 설명**   | **서식 값의 예**     |
|-----------|-------------|-------------------------------------------|
| %d        | 4byte 정수    | \n                                        |
| %x        | 16진법        | printf("숫자를 입력하시오.: %x", 10) <br/> 출력: a  |
| %o        | 8진법         | printf("숫자를 입력하시오.: %x", 10) <br/> 출력: 12 |
| %f        | 실수          |  |
| %lf       | 8byte 실수    |  |
| %u        | 4byte 양수    |  |
| %hi, hu   | 2byte 정수 양수 |  |
| %li, lu   | 4byte 정수 양수 |  |
| %lli, llu | 8byte 정수 양수 |  |
| %llf      | 12byte 실수   |  |
| %e, E     | 부동소수점 e, E  |  |

### 2. 문자의 출력
| **서식** | **서식 설명**  | **서식 값의 예**  |
|------|------------|--------|
| \'   | 문자 출력      |       |
| \"   | 문자열 출력     |        |
| %p   | 포인터 주소값 출력 |      |

## C3. 자리수를 맞춘 출력

**정수**

| **서식** | **서식 설명**                           |
|--------|-------------------------------------|
| %d     | 숫자의 자릿수만큼 출력                        |
| %5d    | 오른쪽에 붙여서 정렬함, 5자리를 확보함              |
| %05d   | 오른쪽에 붙여서 정렬하고 빈칸을 0으로 채우고 5자리 확보  |

**실수**

|-------|----------------|
| %f    | 소수점 여섯 자리까지 출력 |
| %7.1f | 소수점 첫째 자리만 출력  |
| %7.3f | 소수점 셋째 자리만 출력  |

### 문자열형 데이터 서식은 오른쪽에 맞추어 출력됨

|--------|-------------|

| %10s   | 10칸 확보 후 출력 |

# 2주차 2교시: 변수의 이해
## D4. 자료형 개념
정의: C 프로그램은 숫자, 문자, 단어들로 이루어진 데이터를 사용하는데, 이런 데이터의 종류를 자료형이라고 함.
- 컴퓨터는 256개의 문자를 알고 있는데 이 문자들을 아스키 테이블에서 찾을 수 있음
- C 언어에서는 스페이스(띄어쓰기)도 하나의 문자로 인식함.
- `+`는 문자이지만 덧셈도 수행 가능함.

### (1) ASCII
`ASCII`: American Standard Code for Information Interchange

`0` = 문자열 종료  
`32` = Space  
`48` = 0  
`65` = A  
`97` = a  

```
'5' - '0' = 5 // 문자 -> 숫자
```

### (2) 정수
- 정수는 0으로 시작하면 C 언어에서는 16진수 혹은 8진수로 입력됨. (안되는 예: 0123)

### (3) 변수
**정의**: 데이터를 저장하기 위해 프로그램에 의해 이름을 할당 받은 메모리 공간을 의미하고 이렇게 저장된 값은 변경될 수 있음
- 한 프로그램에 여러 데이터를 저장하려면 그에 맞는 변수가 필요함
- C언어에서 숫자 표현에 관련된 변수는 정수형 변수와 실수형 변수로 표현 가능함.

**정수형 변수**
- char 형
- int 형
- long 형

**실수형 변수**
- float 형
- double 형

#### ㄱ. 재밌는 변수 전환
```
int d;
d=97;
printf('%c',d) 
```
출력 결과: a

- 문자열의 끝은 "Null Character"로 되어있음. : 끝을 알기 위함.: 공간 하나 빼줘야함.

#### ㄴ. 변수의 이름
변수의 이름 규칙
- 하나의 프로그램에서 같은 이름의 변수를 사용할 수 없기에 변수의 이름은 달라야함.
- 변수의 이름은 1~31자 사이가 좋음.
- 변수의 첫글자는 반드시 **알파벳**이어야 함. 
- 이후에는 숫자나 밑줄(_)이 가능.

## E5. 변수 선언
### (1) 변수의 이름
- 대소문자를 구분함 
- 변수 이름에 공백을 포함할 수 없음
- 예약어(keyword) 사용 불가능

### (2) 변수의 초기화
- 변수를 선언만 하면 변수에 '쓰레기 값'이 들어가기 때문에, 선언과 동시에 초기화 해줘야함.
- 아래는 초기화의 방법.
```
int number = 0;
```

### (3) 변수 선언
**정의**: 변수는 사용하기 전에 반드시 정의해야 함
- 변수를 선언한다는 것은 C언어에게 변수 공간이 필요하니 예약을 한다는 개념
- 변수를 정의하려면 변수 종류(자료형)를 적고 변수의 이름을 적음,
- 변수의 종류가 같으면 **콤마를 사용하여 연속적으로 표현 가능**함
  - 연속 선언 방법: `int a,b,c,d,e,f;`

## F6. 변수에 값 대입
### (1) 대입 연산자(Assignment Operator)
**정의**: 변수에 값을 저장하라는 의미, C언어에서는 이 대입 연산자를 등호(`=`)로 표현함.  
예: int number = 0;

#### ㄱ. 사용예
```
int number = 10 + 5;
int number = a + b; 
a=b=c=d=100; // a에 100이 들어감
```

## G7. 상수 개념
**정의**: 변하지 않는 데이터를 담아 놓은 공간
- 변수와 비슷하지만, 한번 데이터를 저장하면 프로그램 실행 시 변하지 않는다는 차이점이 있음
      - 사용 예 `const int NUM = 10;`
- 대문자로 표현해야함., 선언과 초기화를 동시에 해야함.

**리터럴 상수**
- 1, 3, a 와 같이 변하지 않는 데이터

**심볼릭 상수**
- 변하지 않는 변수
- `const` 또는 `#define`을 이용해서 선언함
