# 3주차 1교시: 데이터 단위 및 진법 변환
## H8. 비트와 바이트
### (1) 비트
**정의**: 컴퓨터는 전기의 흐름을 디지털화 하여 0과 1로 표현되는 신호에 의해 동작함
- 컴퓨터에서 표현하는 가장 작은 단위로 0과 1이라는 신호로 표현
### (2) 바이트
**정의**: Byte라는 의미는 컴퓨터에서 한 번에 처리하는 데이터의 크기를 의미함.
- 1byte는 8bit
### (3) Nibble
**정의**: 4bit
### (4) 워드
**정의**: 16bit, 하나의 기계어 명령이나 연산을 통해 저장된 장치로서 **레지스터에 옮겨 놓을 수 있는 데이터 단위.**

## J10. 숫자 데이터 형식 개요
### (1) 숫자형 데이터 형식 개념
데이터 형식: 선언할 변수의 종류를 의미함
- C언어에서는 변수들은 사용하기 전에 반드시 정수, 실수, 문자열 등의 데이터 형식을 선언해야함

C언어에서 사용되는 숫자 데이터 형식
- 정수형
- 실수형

정수형 데이터 형식  
- **char**: 1byte
- unsigned char: byte
- **short**: 2byte
- unsigned short: 2byte
- **int**: 4byte   
- unsigned int: 4byte
- **long**: 4byte
- unsigned long: 4byte

다양한 자료형이 제공되는 이유
> 데이터의 표현 방식이 다르기 때문임
- C 언어에서는 데이터 형식의 크기를 byte(바이트) 단위로 구하는 `sizeof` 연산자가 있음
- CPU가 연산하기에 적합한 데이터의 크기가 int형 크기이다.

#### ㄱ. sizeof 표현방법
```yaml
int main()
{
  int sizeofNum = 0;
  
  sizeofNum = sizeof sizeofNum;
  sizeofNum = sizeof(int);
  sizeofNum = sizeof(sizeofNum);
}
```


### (2) 실수형 데이터 형식 개념
