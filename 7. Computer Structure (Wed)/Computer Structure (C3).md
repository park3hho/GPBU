# 3주차 1교시: 
## L12. BCD Code
### (1) 개요
Binary Coded Decimal Code, 2진화 10진코드, 8421 코드

- 표기는 2진수이지만 의미는 10진수
- 1010~1111까지는 사용자히 않음
- 키패드 입력, 디지털 판독 시 표현 방법

### (2) BCD의 종류
**가중치 방식 코드**: 자리마다 가중치(자릿값)
- 8421
- 6311
- 5421
- 5311
- 5211
- 51111

**비가중치 방식 코드**: 자리마다 가중치가 없음
- 3초과 코드
- 그레이 코드
- 2 out of 5 코드

### (3) 8421 코드
#### ㄱ. 표현범위
네자리수로 표현 우리가 아는 표현임. 하지만 네자리마다 10씩 표현까지 가능하기 때문에 `0001 0001` 은 11임.

#### ㄴ. 무효코드
- 4비트로 표현되는 16개의 숫자 중 0부터 9까지만 표현되므로 **사용하지 않는 코드**

#### ㄷ. 8421의 연산
- 그냥 다음거 하나 올리면 됨. 

### (4) 3초과 코드
- 원래 숫자에 3더해서 표현
- 모든 비트가 동시에 0이 되는 경우가 없으므로 신호 두절을 구별할 때 유용함.

**자기 보수의 성질**

### (5) 그레이코드
가중치가 없는 비산술용 코드
- 연속되는 코드들 간에 하나의 비트만 변화하여 코드 생성
- 고속 I/O와 ADC(Analogue Digital Convert)에서 오류를 줄이기 위해 사용

#### ㄱ. 연산방법
2진코드 &lrarr; 그레이코드 (XOR)


## M13. 영문 숫자 코드
### (1) ASCII(American Standard Code for Information Interchange)
컴퓨터나 디지털 시스템에서 사용되는 코드
#### ㄱ.7비트 표현방식: 128개의 문자 조합
- 영대소문자 52개, 숫자 10 특수문자 33개 제어문자 33개 
- 인쇄 가능 문자 32~126개, 33개의 제어문자 0~31, 127

#### ㄴ. EBCDIC Extended Binary Coded Decimal Interchange Code
- 8비트로 256개 표현 가능

## N14. 패리티 비트 오류검출코드
### (1) 패리티 검사
정보 비트 수가 적고, 오류 발생 확률이 낮은 경우, 주로 사용하는 오류 검출 방식
패리티 검사의 특징
- 오류검출 가능, 오류정정 불가능
- 어느 비트에 오류가 발생했는지 알 수 없고, 짝수 개의 오류가 발생 시 오류 검출이 불가능
- 구현이 간단, 비동기 통신에 많이 사용함

### (2) 패리티 검사 비트
- 짝수 패리티: 데이터 1의 개수를 짝수 개로 맞춤
- 홀수 패리티: 데이터 1의 개수를 홀수 개로 맞춤

## O15. 순환 중복 검사(CRC, Cucle Redundancy Check)
- CD, DVD등에 사용함
- 1~2개의 비트 오류를 검출하는 방법

### (1) CRC 계산에 사용되는 모듈로 -2 연산
- XOR 연산이다.
- 발생코드로 확인할 수 있음.

### (2) 키는 생성 다항식이다
- $x^7$ x $x^5$ x $x^3$ x $x^0$

## P16. 해밍 코드
특징: 단말기간 오류 검출, 정정할 수 있는 코드
- 짝수 패리티 비트 사용

### (1) 해리티 비트와 데이터 비트의 관계
- 0000은 에러가 없음을 의미함.

## Q17: 기본 논리게이트의 개념과 종류
### (1) TTL과 CMOS 논리 레벨 정의영역
**TTL**: (2.5~5V) &lrarr; (정의되지 않은 영역) &lrarr; (0~0.8V)  
**CMOS**: (3.5~5V) &lrarr; (정의되지 않은 영역) &lrarr; (0~1.5V)  

### (2) NOT 게이트
- 입출력이 하나
- 보수로 출력됨 (반대로)
- 세모랑 버블
- IC7404
- 스위치 회로
- 트랜지스터 회로

### (3) Buffer 게이트
- 잠시 지연
- 세모만
- 입력 A, 제어단자 E, 출력 F

### (4) AND 게이트
- 2 입력
- 논리곱
- 곱셈이라 보면 됨.
- IC 7408
- IC 7411

### (5) OR 게이트
- 2 입력
- 논리합
- IC7432
- CMOS4075
