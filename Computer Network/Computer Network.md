======= 1주차 1교시 =======

1. 인류 사회 변천과 컴퓨터 네트워크
1) 제1의 물결(농업혁명) -> 제2의 물결(산업혁명) -> 제3의 물결(정보산업혁명) -> 제4의 물결(유비쿼터스 혁명)
2) 공간 혁명의 역사: 도시혁명 -> 산업혁명 -> 정보혁명 -> 유비쿼터스 혁명

2. 컴퓨터 네트워크 기술의 개념
정보통신 기술(ICT) = 정보 처리 기술(IT) + 통신 기술(CT)
- 데이터를 의미 있는 정보로 변환하거나 발전하여 데이터를 지능 또는 지식으로 변환하기 위해 컴퓨터를 이용하여 처리하는 것.
- 여러 단말 장치를 통신망에 접속하여 데이터를 전송하고 처리하며 교환하는 통신 체계
- 좁은 의미로 데이터 통신

통신 기술: 멀리 떨어진 정보원과 정보 목적지 사이에서 정보를 전송하고 처리하는 기술

ㄱ. 정보처리 기술 절차
(1)데이터 발생; (2)데이터 수집; (3)데이터 분석; (4)데이터 처리; (5)데이터 저장 및 활용;

ㄴ. 정보처리 기술(ICT)
데이터: 현실세계를 단순히 관찰, 측정하여 수집하거나 생산한 사실이나 측정치
정보: 데이터를 가공하여 변환하여 얻은 결과물; 유요하게 해석하거나 데이터의 상호관계 등 의사결정 하는 데 도움이 되는 가치 있는 데이터.
지식: 지식은 교욱과 학습을 통해 사람이 재활용 할 수 있는 정보와 기술 등
컴퓨터: 방대한 양의 데이터를 신속하고 정확하게 처리하고 부넉하여 사용자에게 유용한 결과를 제공; 데이터 처리 시스템


3. 컴퓨터 네트워크 기술의 탄생과 발전 과정
ㄱ. 통신(정보전송) 기술의 탄생과 발전과정
전기통신 이전의 통신(원시통신)
- 몸동작이나 언어, 각종 물리적 도구를 이용해 통신
전기통신
- 문자나 음성 등의 데이터를 전기적 신호로 변환한 후 이를 전송매체를 이용해 전송
- 제1세대: 전신
- 제2세대: 전화
정보통신
- 제3세대: 데이터 통신
- 제4세대: 정보통신(통신+컴퓨터 기술 융합)

ㄴ. 전기통신 시대
1) 제1세대: 전신
- 최초의 전기통신이자 제1세대 통신
-1937년 미국의 모스가 발명한 방식

사무엘 모스(1791~1872)
- 미국의 발명가면서 화가
- 1844년 미국의 전신 실용화를 추진
- 워싱턴-볼티모어 간 60km의 전화 회선 설치, 통신에 성공
- 1825년에 부인의 장례식에도 참가할 수 없었던 사건이 개인에게 큰 영향을 끼침
- 1932년 모스 부호를 착안

2) 제2세대: 전화
- 1876년 미국의 그레이엄 벨이 최초로 발명
- 그 뒤 1878년 에디슨이 탄소식 전화기 개발

알렉산더 그레이엄 벨(1847~1922)
- 영국 태생의 미국 과학자-발명가
- 1871년 음성의 메커니즘에 흥미를 느끼다가 전기통신을 연구
- 1873년 보스턴 대학교 음성생리학 교수임용

3) 제3세대: 데이터 통신
- 컴퓨토를 이용한 통신 체계와 비슷
1946년 - 에니악 개발
1950년 - 사무용 전자계산기 개발
1958년 - 세계 최초의 정보통신 시스템 SAGE
1961년 - 좌석 예약 시스템 개발

4) 제4세대: 정보통신
- 컴퓨터로 통신을 공유하는 형태
- 자원 등을 공유하고 통신회선을 효율적으로 이용하여 비용을 절감하려는 목적에서 시작.

1971년 - 세계 최초의 패킷 교환망, ARPA
1975년 - 상업용 패킷 교환망, TELENET, TYMNET
1980년 - 디지털 기술의 VAN, ISDN
1990년 - 위성통신을 이용한 데이터 통신과 이동통신, 인터넷 보급과 웹 등장
2000년 - 인터넷, 웹 이동통신(스마트폰 포함) 확산

5) 종합정보통신망(ISDN)
- 모든 서비스를 단일 통신망으로 통합
- 이를 시작으로 광대역 융합망, 초광대역 융합망 등으로 발전

6) 유비쿼터스 컴퓨팅
정의: 수많은 지능형 컴퓨터를 유무선 네트워크와 연결하여 필요한 정보를 언제, 어디서나, 즉시 제공받는 환경이나 세계를 뜻함.

사물 인터넷의 구현 기술
- 센싱기술
- 유무선통신: 고속처리와 병렬처리를 할 수 있는 유무선 통신 및 네트워크 인프라 기술
- 서비스 인터페이스 기술
- 보안기술


======= 1주차 2교시 =======

1. 네트워크 기초 용어
> 네트워크: 전송매체로 서로 연결된 시스템의 모음
> 시스템: 내부 규칙에 따라 능동적으로 동작하는 대상
> 인터페이스: 시스템과 전송 매체의 연결지점에 대한 규격
> 전송매체: 시스템끼리 데이터를 전달하기 위한 물리적인 전송 수단
> 프로토콜: 전송매체를 통해 데이터를 교환할 때 임의의 통신 규칙
> 인터넷: 전 세계의 네트워크가 유기적으로 연결되어 동작하는 통합 네트워크 
> 표준화: 서로 다른 시스템이 상호 연동해 동작하기 위한 통일된 연동 형식

시스템 기초 용어
- 노드: 인터넷에 연결된 시스템의 가장 일반적인 용어
- 호스트: 컴퓨팅 기능이 있는 시스템
- 클라이언트: 서비스를 요청하는 시스템
- 서버: 서비스를 제공하는 시스템

2. 네트워크 개념과 기능
사전적 의미: 모뎀이나 LAN, 케이블, 무선매체 등 통신설비를 갖춘 컴퓨터로 서로 연결하는 조직이나 체계, 통신망
- 통신설비들로 두 대 이상의 컴퓨터를 서로 연결한 것
- 컴퓨터 두 대로 연결했든, 그 이상으로 연결했든 간에 필요에 따라 여더 대를 서로 연결한 것이 네트워크

ㄱ. 
네트워크의 기능
Layer7: 응용 계층
Layer6: 표현 계층
Layer5: 세션 계층
Layer4: 전송계층
Layer3: 네트워크 계층
Layer2: 데이터링크 계층
Layer1: 물리 계층

네트워크 계층 - IP 프로토콜
전송 계층 - TCP, UDP 프로토콜

3. 인터네트워킹
정의: 그 말이 함유하는 뜻대로 하나 이상의 망을 상호 연결하는 것
- 단순히 물리적으로 망을 연결한다는 의미 의외에도 망을 논리적으로 연결한다는 것을 의미
REAL 정의: 사용자로 하여금 인터네트워킹의 존재를 인식하지 않고 마치 하나의 망을 사용하는 것처럼 느끼게 해주는 것

게이트웨이
- 인터네트워킹 기능을 수행하는 시스템
> 리피터: 물리 계층을 지원 (신호 증폭)
> 브리지: 물리 계층과 데이터 링크 계층을 지원
> 라우터: 물리 계층, 데이터 링크 계층, 네트워크 계층을 지원

데이터 단위
APDU: 응용 계층의 데이터 단위
PPDU: 표현 계층의 데이터 단위
SPDU: 세션 계층의 데이터 단위
TPDU: 전송 계층의 데이터 단위
NPDU: 네트워크 계층의 데이터 단위
DPDU: 데이터링크 계층의 데이터 단위


3. 네트워크 주소의 표현
1) IP 주소
- IPv4 프로토콜 사용
- 32 비트 크기의 주소 체계
- IPv6에서는 128 비트 주소 체계로 확장

2) 라우팅
- 인터넷 상에서 패킷의 이동 경로 선택 해주는 것

3) 호스트 이름
- 네트워크 상의 한 노드를 유일하게 식별하는 수단
- 호스트 이름
  - 영문자 형식
  - 사람에게 기억하기 쉽고 친숙한 가변길이의 명칭
  - 대소문자 구분 안함
- 호스트 주소: 숫자 형식
  - 기계장치가 네트워크 상에서 호스트를 쉽게 찾도록 통상적으로 사용하는 고정길이의 기계적 주소
