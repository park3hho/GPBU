# 자료구조 과제

- **과제주제**: 최근 4차 산업혁명시대의 필수 중 데이터터베이스 기술이 미래 산업의 중요한 부분을 떠오르고 있다 .빅데이터의 개념과 주요 기술에 대하여 정리하고 빅데이터 기술이 앞으로 어떻게 변화되고 발전할지에 대하여 의견을 제시하시오.
- **부연설명**: 
  - 서론: 빅 데이터 정의 및 특징에 대해 설명
  - 본론: 빅 데이터의 주요 핵심 기술에 대해 설명
  - 결론: 빅데이터가 앞으로 어떻게 발전할 것인지에 대한 스스로의 생각을 정리하여 기술
